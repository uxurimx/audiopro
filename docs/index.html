Cinema Mode

Un archivo.
Cada quien su idioma.

audifonospro detecta todas las pistas de audio de tu archivo MKV o MP4 y las distribuye simultáneamente a cada dispositivo conectado. Cada persona elige qué pista escucha. Nadie pierde calidad.

  • ✓ Soporte para MKV, MP4, AVI, WebM con múltiples pistas de audio

  • ✓ EQ independiente por dispositivo para nivelar diferencias de calidad
  • ✓ Video en pantalla con sincronización de audio garantizada

🎧 Audífonos

Cualquier micrófono. Sin restricciones.

El Traductor detecta automáticamente todos los micrófonos disponibles en PipeWire. ¿Tienes JBL con mic? Activa el modo HFP con un clic y el sistema cambia el perfil Bluetooth, espera la reconexión y selecciona el mic automáticamente.

¿Prefieres el micrófono de tu laptop? También funciona. ¿Quieres cancelar el ruido del ambiente antes de transcribir? Activa ANC con un toggle y el sistema crea un micrófono virtual limpio en PipeWire.

Auto-detección de idioma

Whisper detecta automáticamente el idioma que hablas. No necesitas configurar nada. Habla español, inglés, francés o japonés — el sistema lo reconoce y traduce al idioma destino que elijas.

ANC — Cancelación de Ruido

Silencio de $400.
Gratis.

Los audífonos con ANC de hardware cuestan una fortuna. audifonospro implementa cancelación de ruido por software con 5 niveles de intensidad — desde filtrado espectral hasta ANC adaptativo LMS con el micrófono de tu laptop como referencia.

0Sin ANC
1Espectral
2RNNoise
3LMS
4+GATT

El nivel 4 controla el ANC de hardware de tus JBL directamente vía BLE GATT — sin aplicación del fabricante, sin Bluetooth LE restricciones, directamente desde audifonospro.

🧠

WebRTC ANC integrado en PipeWire

Usa libspa-aec-webrtc de PipeWire para crear un micrófono virtual limpio. El mismo algoritmo de cancelación de eco que usa Google Meet, corriendo localmente en tu laptop.

⚡

Filtro LMS adaptativo

El Least Mean Squares usa el micrófono de la laptop como referencia del ruido y genera la señal anti-ruido en tiempo real. Se adapta al entorno — funciona en cafeterías, oficinas y transporte público.

Tu privacidad, en tu máquina.

Todo el procesamiento ocurre en tu laptop. Sin APIs externas obligatorias, sin datos enviados a terceros, sin cuentas requeridas. La nube es opcional.

Arquitectura del pipeline de traducción

── Hilo de captura ────────────────────────── GStreamer pulsesrc (mic JBL / laptop / ANC) → appsink (chunks 16kHz S16LE) → VAD (energía RMS + zero-crossing) → Queue[raw] (maxsize=10) ── Hilo STT ────────────────────────────────── Queue[raw] → Whisper.cpp / OpenAI → segmento de texto → Queue[segment] (maxsize=3) ── Hilo de traducción ─────────────────────── Queue[segment] → Ollama / GPT-4o-mini → texto traducido → Queue[tts] (maxsize=5) ── Hilo TTS ────────────────────────────────── Queue[tts] → Piper / edge-tts / OpenAI → audio bytes → pulsesink (dispositivo destino)

La historia

audifonospro nació de una necesidad real: una noche de cine con familia que habla diferentes idiomas. La solución obvia — múltiples dispositivos con diferentes pistas de audio — no existía de forma simple en ningún sistema operativo.

Lo que empezó como un script para enrutar audio en PipeWire creció en un sistema completo. La segunda pregunta fue: si ya tengo el audio capturado y ruteado, ¿por qué no traducirlo también? Así llegó el Traductor. Y si ya puedo traducir, ¿por qué no aplicar cancelación de ruido antes? Así llegó el ANC.

Cada problema resuelto reveló el siguiente. El resultado es un sistema que trata el audio de tu laptop como lo que realmente es: infraestructura personal crítica que merece el mismo nivel de control y personalización que cualquier herramienta profesional.

Construido con Python, GTK4, GStreamer y PipeWire. Desarrollado en Fedora 43. Diseñado para durar. Construido para extenderse.
